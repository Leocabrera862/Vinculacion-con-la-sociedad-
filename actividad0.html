Actividad 

Indicaciones

En esta actividad, se llevará a cabo un análisis detallado del laberinto. Posteriormente, se utilizará mBlock para implementar los comandos necesarios que permitirán resolver el laberinto. Finalmente, se redactará un informe que describa el proceso de análisis realizado y los pasos seguidos en la programación.

El tiempo estimado para realizar esta actividad es de 30 minutos.
